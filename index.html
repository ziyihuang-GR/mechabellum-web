单位排名｜当期强度 / 长期强度

运算公式
当期强度
Si,t = ∑k∈K ωk · Pi,tk
K = { 总胜利排名，最强科技排名，平均科技排名，科技携带排名，最强组合排名，平均组合排名，增援排名，最强装备排名 }
Pi,tk = (胜场百分位i,tk)α · (胜率百分位i,tk)1−α
长期强度
Li,T = ∑t≤T Si,t · βq(t,T) · γMNM(t,T) · γmNm(t,T)
q(t,T) = 当前期数 − 历史期数
NM(t,T) = # { 重大更新点 u : t < u ≤ T }
Nm(t,T) = # { 小型更新点 u : t < u ≤ T }
时间轴：-
当期强度使用右端时间；长期强度使用左端到右端的时间范围。
